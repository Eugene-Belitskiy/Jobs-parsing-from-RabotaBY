# Тестовая версия парсера

`test_main.py` — упрощенная версия для быстрой проверки работоспособности парсера перед полным запуском.

---

## Ограничения тестового режима

| Параметр | Полная версия | Тестовая версия |
|---|---|---|
| Специализации | 174 | 1 (первая из списка) |
| Страниц поиска | все | 1 |
| Вакансий | все | 5 |
| Время выполнения | несколько часов | ~1-2 минуты |
| Префикс файлов | — | `TEST_` |

---

## Запуск

```bash
python test_main.py
```

---

## Что проверяет тест

1. **Инициализация Chrome** — браузер запускается и обходит защиту rabota.by
2. **Сбор ссылок** — парсер находит вакансии на первой странице поиска
3. **Парсинг вакансии** — все поля корректно извлекаются из страницы вакансии
4. **Гармонизация** — зарплата, опыт, город, категория — всё нормализуется
5. **Сохранение** — JSON создается в папке `data/`

Дополнительно запускается **проверка извлечения зарплаты** на 4 тестовых вакансиях с известными значениями.

---

## Выходные файлы

После выполнения в `data/` появятся:

```
data/
├── TEST_links_and_names_MM.YYYY_rabota_by.json
├── TEST_url_list_MM.YYYY_RabotaBy.txt
└── TEST_data_finally_MM.YYYY_Rabota_by.json
```

Структура JSON полностью совпадает с основной версией — те же 26 полей.
