# Конфигурация специализаций

В этой папке хранятся два файла, которые определяют, **какие вакансии собирать**.

---

## Файлы

### `search_links.txt`

Ссылки поиска на rabota.by — по одной на строку. Строки с `#` и пустые строки игнорируются.

Уже содержит **174 ссылки** по всем профессиональным ролям на рынке труда Беларуси (параметр `area=1002`).

```
https://rabota.by/search/vacancy?area=1002&professional_role=4&...
https://rabota.by/search/vacancy?area=1002&professional_role=8&...
```

### `specializations.txt`

Названия специализаций — в том же порядке, что и ссылки. Используются для поля `Специализация` в конечном JSON.

```
Автомойщик
Автослесарь, автомеханик
Мастер-приемщик
```

**Важно:** количество строк в обоих файлах должно совпадать.

---

## Как добавить свою специализацию

1. Откройте [rabota.by](https://rabota.by) и найдите вакансии по нужному запросу
2. Настройте фильтры (если нужно)
3. Скопируйте URL из адресной строки
4. Добавьте URL в конец `search_links.txt`
5. Добавьте название в конец `specializations.txt`

**Пример:**

`search_links.txt`:
```
https://rabota.by/search/vacancy?text=data+engineer&area=1002
```

`specializations.txt`:
```
Инженер данных
```

## Тестирование

Чтобы проверить, что новая ссылка работает, поставьте её первой в файле и запустите:

```bash
python test_main.py
```

Тест обрабатывает только первую специализацию — 1 страница, 5 вакансий.
